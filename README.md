A fast, gothic character sheet app for MORK BORG. Build characters, track play, and keep everything in your browser.

## Features

- Classless random character generation with overwrite confirmation.
- Full sheet editor for Identity, Core Stats, Survival, Equipment, Powers, and Notes.
- Multi-character vault with quick switching.
- Auto-save to browser storage and JSON import/export for backups.
- Sinner/Saint theme toggle with persisted preference.
- Interactive HP tracker with low-HP styling and `BROKEN` state handling.
- Custom weapon and armor support with editable die values and flexible attack stat selection.
- Powers workflow with cast attempts, known powers list, scroll counters, and notes.
- In-app quick-reference rules and tables for fast at-the-table lookups.
- Built-in collapsible dice tray with 3D dice, history, and reroll support.
- Lazy-loaded dice module with `Force 2D` and `Low Performance` modes for broader device compatibility.
- Works offline after first successful load (PWA install supported).
- Responsive layout for desktop and mobile.


## Privacy

This app stores character and dice data in your browser (`localStorage`). No backend database is required.
